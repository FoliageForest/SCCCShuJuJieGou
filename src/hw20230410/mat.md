`mat[i][?]` 对应前补制表符的数量为 

| i | 数量 |
| --- | --- |
| length - 1 | 0 |
| length - 2 | 1 |
| ... | ... |
| i | length - i - 1 |
| ... | ... |
| 1 | length - 2 |
| 0 | length - 1 |
